# APK Sentinel Frontend

The frontend provides an interactive interface for uploading Android APK files and visualizing the security analysis results generated by the backend.

---

## Features

- Modern Responsive UI
- APK Upload
- Drag and Drop Support
- Risk Score Visualization
- AI-generated Security Summary
- Dangerous Permission Display
- Suspicious API Display
- URL Detection Results
- Error Handling

## Installation

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

## Build Production Version

```bash
npm run build
```

---

## Preview Production Build

```bash
npm run preview
```

## Backend Connection

The frontend communicates with the FastAPI backend running on:

```
http://127.0.0.1:8000
```

Ensure the backend server is running before starting the frontend.
